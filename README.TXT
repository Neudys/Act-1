----------------PASOS PARA LA EJECUCION DE EL PROYECTO----------------
----------------1- verificar versiones----------------



                    node -v  
                    npm -v  


----------------2- instalar vite----------------



                npm init vite

                    nombre: e-piphany
                    tipo: vanilla
                    lenguaje: javascript

                cd e-piphany
                npm install



----------------3- instalar otras dependencias (leni , locomotive, gsap)----------------



                npm install gsap @studio-freight/lenis locomotive-scroll



----------------4- abrir en local----------------




                npm run dev
